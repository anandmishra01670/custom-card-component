# How to run the project
### Prerequisites
* Node.js
* npm

### Verify Node.js 
- node -v

### Setup and Run
Open a terminal (Command Prompt / PowerShell / Git Bash / Terminal) and follow the steps below.
1. Clone the repository
* git clone https://github.com/anandmishra01670/custom-card-component.git
2. Navigate to the project
* cd custom-card-component
3. Install dependencies
* npm install
4. Start the server
* npm run dev
5. After running the command, Vite will display a local URL in the terminal.
* Open that URL to access the GrapesJS editor
